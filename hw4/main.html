minimal command-based live coding engine

how this works:

  • separate beats with spaces
  • for each beat, specify [length]@[pitch]. length and pitch can be javascript computation values
  • if you want to optionally do FM modulation, add f[modulationIndex],[modulationFrequency] after the pitch

    • for example: 1+1@220 2@220*2f200,220*3 1*3@270

    press the button to re-evaluate the code.
